## Description
The Port Scanner module scans a specified target IP address or hostname to detect which network ports are open from port 1 to 1024. Open ports potentially indicate services running on the host that might be exploited. It uses low-level socket programming to attempt connections on ports within the specified range and lists the ports responding affirmatively.
The Brute Forcer module attempts to gain unauthorized access to a web login form by systematically trying passwords from a user-supplied list for a given username. It sends HTTP POST requests to the login URL and analyzes the serverâ€™s response to determine if the login was successful. This simulates a brute-force attack, revealing weak passwords and highlighting authentication vulnerabilities.
